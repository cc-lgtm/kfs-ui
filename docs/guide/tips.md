# Tips
#### 默认
:::demo
```vue
<template>
  <div>
    <cc-tips tips="js是浏览器寄生语言">摸我一下</cc-tips>
  </div>
</template>
```
:::

#### 属性
| 属性 | 描述 | 类型 | 可选值 | 默认 |
| ------------- |:-------------:| -----:|:-------------:| -----:|
| class | 自定义tips的样式类名 | `string` | `-` | `-`  |
| tips | 提示的文字 | `string` | `-` | `-` |
| color | 提示的文字颜色 | `string` | `-` | `#57606f` |

#### 事件
| 名称 | 描述 | 参数 |
| ------------- |:-------------:| -----:|
| hover | 鼠标悬停触发事件 | `-` |
| hoverout | 鼠标离开触发事件 | `-` |

#### slots
| 名称 | 描述 |
| ------------- |:-------------:|
| `-` | `Tips`内自定义内容 |
